# DIY-java-datastructures

Hjemmelagde databeholdere, uten bruk av java sine innebygde liste biblioteker. Jeg har implementert fem forskjellige typer lister; stack (Stabel.java), queue (Koe.java), indexed list (IndekstertListe.java), og priority queue (PrioritesKoe.java). 

### Stack - Last in, First out (LIFO)
Stacks, eller stabler, er lister der nye elementer legges til først i listen, og fjerning skjer første element i listen.

### Queue - First in, First out (FIFO)
I en queue, eller kø, settes nye elementer inn bakerst og henter dem ut fra starten.

### Indexed List
I en indeksert liste så hentes og fjernes elementene som blir spesifisert med en index, som i java sine kjente arraylists.

### Priority queue
I prioritetskøer haster noen elementer mer enn andre;
de tas først uavhengig av hvor lenge de har stått i køen. Høyeste prioritet er laveste verdi.
